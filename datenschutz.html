Datenschutzerklärung

Allgemeines

Diese Website dient ausschließlich der Anzeige öffentlich verfügbarer Konzert- und Veranstaltungsinformationen.

Es werden keine personenbezogenen Daten von Besucher:innen erhoben.

Hosting über GitHub Pages

Diese Website wird über GitHub Pages gehostet.

Beim Aufruf der Website können durch den Hostinganbieter technische Zugriffsdaten verarbeitet werden, insbesondere:

  • IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • Browserinformationen
  • Betriebssystem
  • Referrer-URL

Weitere Informationen: GitHub Privacy Statement

Web Analytics über GoatCounter

GoatCounter ist eine OpenSource Alternative zu Google Analytics und Co., verwendet keine Cookies und speichert keine personenbezogenen Daten. Für weitere Informationen siehe: GoutCounter Privacy Statement

Keine Cookies / kein Tracking

Diese Website verwendet keine Cookies, kein Webtracking und keine Analyse-Tools.

Automatisierte Erfassung von Veranstaltungsdaten

Die dargestellten Veranstaltungsinformationen werden automatisiert aus öffentlich zugänglichen Webseiten von Veranstaltungsorten erfasst.

Erfasst werden ausschließlich öffentlich veröffentlichte Informationen:

  • Veranstaltungsname
  • Künstler:innenname
  • Veranstaltungsort
  • Datum und Uhrzeit
  • Beschreibungstexte

Die Daten dienen ausschließlich der aggregierten Darstellung von Konzert- und Veranstaltungsinformationen.

Sollte ein Veranstalter oder Rechteinhaber eine Entfernung bestimmter Inhalte wünschen, genügt eine formlose Kontaktaufnahme per E-Mail.
